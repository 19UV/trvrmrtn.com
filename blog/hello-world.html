Now, because I lack the funds to buy a plane ticket to Norway, I figured I may as well make a website.

And because I didn't feel like learning a new framework, I figured HTML would be suffice.

But before I actually post anything, I want to lay down some ground rules:


The Blogger's Manifesto

This is a blog, and it should be nothing else.

It shouldn't be fancy, it just needs to work.

It shouldn't be pretty, as long as it's effective.

It shouldn't follow best practices, as long as it works.

It shouldn't be big, because bandwidth is precious.

It shouldn't fast, because caching is the root of all evil.

Most importantly: It shouldn't cost time.


This shouldn't be an ongoing project that requires maintinence. It should just work. I should be able to just post to it.

Consiter this a promise to keep things simple.


That's everything for today,

Cheers.
